# APC-2026-1-
Este repositório foi criado para organizar os conteúdos, exercícios, listas e projetos desenvolvidos na disciplina de Algoritmos e Programação de Computadores (APC) no semestre 2026-1.
A disciplina tem como objetivo desenvolver o raciocínio lógico, a resolução de problemas e a base da programação utilizando a linguagem Python, preparando o estudante para disciplinas mais avançadas da área de computação
Minhas expectativas para o final da matéria são conseguir dominar os conceitos básicos de programação, melhorar meu raciocínio lógico e ser capaz de resolver problemas de forma independente.
Espero também passar na disciplina com um bom desempenho, já que tive dificuldades anteriormente, e sair com uma base sólida em Python para continuar evoluindo nas áreas de Algoritmos, Estruturas de Dados e Cloud.
Além disso, quero terminar o semestre com confiança para programar, criar pequenos projetos e me sentir preparado para as próximas disciplinas da UnB.
