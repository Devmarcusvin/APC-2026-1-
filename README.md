# APC-2026-1-
Este repositório foi criado para organizar os conteúdos, exercícios, listas e projetos desenvolvidos na disciplina de Algoritmos e Programação de Computadores (APC) no semestre 2026-1, turma 07, ministrada pelo professor Jorge Henrique na Universidade de Brasília. A disciplina tem como objetivo desenvolver o raciocínio lógico, a resolução de problemas e a base da programação utilizando Python, preparando o estudante para disciplinas mais avançadas da área de computação. Minhas expectativas para o final da matéria são dominar os conceitos básicos de programação, melhorar meu raciocínio lógico e ser capaz de resolver problemas de forma independente, além de passar na disciplina com um bom desempenho, construir uma base sólida em Python e terminar o semestre com confiança para programar e seguir evoluindo nas próximas disciplinas da UnB.
