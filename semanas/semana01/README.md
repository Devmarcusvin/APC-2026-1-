
# Semana 01

## Objetivos da Semana

Durante a primeira semana, foram realizadas atividades introdutórias voltadas ao desenvolvimento do raciocínio lógico e aos conceitos básicos de programação.

## OctoStudio

Foi realizada uma exploração da plataforma **OctoStudio**, ferramenta educacional voltada para a criação de projetos interativos por meio de programação em blocos. A atividade permitiu compreender conceitos como:

- Sequência de instruções;
- Lógica de programação;
- Eventos e interatividade;
- Desenvolvimento de aplicações simples.

## LMC (Little Man Computer)

Também foram desenvolvidas atividades utilizando o **LMC (Little Man Computer)**, um simulador utilizado para compreender o funcionamento básico de um computador e a execução de instruções em linguagem de máquina.
